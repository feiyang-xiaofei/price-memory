PRIVACY POLICY — PRICE MEMORY (Chrome Extension)
Effective Date: March 2025

------------------------------------------------------------------------

1. INTRODUCTION

Price Memory is a Chrome browser extension that helps you track product
prices as you shop online. This privacy policy explains what data the
extension collects, how that data is handled, and your rights regarding
that data.

The short version: Price Memory stores everything locally on your device.
No data ever leaves your browser. There are no servers, no accounts, no
analytics, and no third parties involved.

------------------------------------------------------------------------

2. WHAT DATA IS COLLECTED

When you visit a product page on a supported shopping site, Price Memory
may read the following information from that page:

  - Product title
  - Product price
  - Product image URL
  - Retailer name
  - Product category (when available)

This information is extracted from the web page you are actively viewing
so that Price Memory can track price changes over time and notify you of
price drops.

Price Memory does NOT collect:

  - Personal information (name, email, address, etc.)
  - Browsing history or URLs beyond the product data listed above
  - Payment or financial information
  - Device identifiers or fingerprints
  - Location data
  - Any form of analytics, telemetry, or usage statistics

No user account or sign-up is required to use Price Memory.

------------------------------------------------------------------------

3. HOW YOUR DATA IS USED

The product data described above is used exclusively to:

  - Display your tracked products and their price history
  - Compare current prices against previously recorded prices
  - Send you local browser notifications when a price drops
  - Show a small floating price badge on product pages you have tracked

That is the complete list. There are no secondary uses.

------------------------------------------------------------------------

4. DATA STORAGE

All data collected by Price Memory is stored locally on your device
using Chrome's built-in storage API (chrome.storage.local). This means:

  - Your data lives entirely within your Chrome browser profile.
  - Your data is never transmitted to any external server.
  - Your data is never uploaded to the cloud by Price Memory.
  - If you use Chrome Sync, Chrome may sync extension storage data
    across your signed-in devices according to your own Chrome settings.
    This is a Chrome platform feature and is not controlled by
    Price Memory.

If you uninstall the Price Memory extension, Chrome automatically
deletes all data associated with it from your local storage.

------------------------------------------------------------------------

5. THIRD PARTIES

Price Memory does not share, sell, rent, or transmit your data to any
third party. Period.

  - No third-party analytics services (no Google Analytics, etc.)
  - No third-party APIs or external services
  - No advertising networks
  - No data brokers

The extension operates entirely offline with respect to its own
functionality. The only network activity is your normal browsing, which
Price Memory does not intercept or modify.

------------------------------------------------------------------------

6. CHROME PERMISSIONS EXPLAINED

Price Memory requests the following browser permissions:

  storage — Used to save your tracked product data locally within Chrome.

  alarms — Used to schedule periodic price checks so the extension can
  detect price changes over time.

  notifications — Used to alert you through Chrome's notification system
  when a tracked product's price drops.

  activeTab — Used to read product information from the page you are
  currently viewing when Price Memory is active.

  Host permissions (all URLs) — Required so that Price Memory can detect
  and extract product information on any shopping website, not just a
  predefined list. The extension does not modify page content except to
  display a small floating price badge on pages where you are tracking
  a product.

These permissions are used solely for the functionality described in
this policy. No permission is used to collect data beyond what is listed
in Section 2 above.

------------------------------------------------------------------------

7. SECURITY

Because all data is stored locally in your browser and is never
transmitted over a network by Price Memory, the primary security
boundary is your own device and browser profile. We recommend:

  - Keeping your Chrome browser up to date.
  - Using a device password or screen lock.
  - Being cautious about who has physical access to your device.

Price Memory does not implement its own encryption layer because the
data never leaves the local Chrome storage environment. Chrome manages
the security of its own storage APIs as part of the browser platform.

------------------------------------------------------------------------

8. CHILDREN'S PRIVACY

Price Memory does not knowingly collect any personal information from
anyone, including children under the age of 13. Since the extension
collects no personal data and requires no account creation, there is no
mechanism through which a child's personal information could be gathered
or stored.

------------------------------------------------------------------------

9. CHANGES TO THIS POLICY

If this privacy policy is updated, the revised version will be published
alongside the extension listing on the Chrome Web Store and on the
product page. The "Effective Date" at the top of this document will be
updated to reflect the date of the most recent revision.

Continued use of Price Memory after a policy update constitutes
acceptance of the revised terms.

------------------------------------------------------------------------

10. DATA DELETION

You can delete all data stored by Price Memory at any time by:

  - Using any built-in "clear data" or "delete" features within the
    extension interface to remove individual tracked products.
  - Uninstalling the extension, which causes Chrome to automatically
    remove all associated local storage data.
  - Clearing your browser's extension data through Chrome's settings
    (Settings > Privacy and Security > Clear Browsing Data).
